# Crypto Guard <!-- omit in toc -->

Консольное приложение для шифровки и хэширования файлов на основе библиотеки OpenSSL.  Предлагает следующие возможности:

* Шифрование и дешифровка файлов с помощью симметричного алгоритма AES-256-CBC;
* Хэширование файлов с помощью алгоритма SHA-256;

## Обзор поддерживаемых команд

* `--command`/`-c` - исполняемая команда, выбор из трёх возможностей: `encrypt` - шифровка, `decrypt` - дешифровка, `checksum` - хэширование;
* `--input`/`-i` - входящий файл;
* `--output`/`-o` - исходящий файл (не требуется при хэшировании);
* `--password`/`-p` - пароль (не требуется при хэшировании);
* `--help`/`-h` - помощь (на английском);
